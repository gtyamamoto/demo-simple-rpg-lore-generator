# Game Horror generative Horror character
A game where it randoms or with some input from user ( a phrase)
will create a animated Character with some random generative agent to create the image, another generative to coordinate the alghoritim to animate it and enable a chat a like interaction with the character
- first only static image
- then animation
- then soundful interaction
- goal game, where you need to investigate how to destroy/kill the horror entity, in 5 or so questions
- agents to build a logic behind of the character weakness and strong to achieve the goal
